CityLex: a free English lexical database location_city

A comprehensive tool for exploring English words and their properties.

Data sources

  • Frequencies:

    • SUBTLEX-UK (CC BY-NC 4.0)
      • Raw frequency
      • Frequency per million words
      • log10 probability
      • Zipf scale
    • SUBTLEX-US (CC BY-NC 4.0)
      • Raw frequency
      • Frequency per million words
      • log10 probability
      • Zipf scale

  • Morphological features:

    • UDLexicons [Apertium] (GNU GPL v3)
      • Universal Dependency-style tags
      • UniMorph-style tags
      • CELEX tags
    • UniMorph (CC BY 4.0)
      • Universal Dependency-style tags
      • UniMorph-style tags
      • CELEX tags

  • Morphological segmentations:

    • ELP (CC BY-NC 4.0)
      • Segmentation
      • Number of morphs

  • Pronunciations:

    • WikiPron-US (Apache 2.0)
      • IPA
    • WikiPron-UK (Apache 2.0)
      • IPA

    • Output Format

    Long TSV [one record per line] Wide JSON [join on word]
